# Nim → Amiga 500 🕹️

> Write modern Nim code that runs on a Commodore Amiga 500 (1985)

![Demo](screenshots/demo.png)

## ✨ Why does this exist?

Because we love the Amiga 500, but miss writing comfortable, modern code.

This project brings **Nim** — a fast, expressive, Python-like language — to the Motorola 68000 architecture, with a runtime so small it fits in the Amiga's limited memory.

## 🚀 Try it in 30 seconds

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/nim-amiga
cd nim-amiga

# Build a test program
./build_amiga.sh examples/hola.nim

# Result: a file called 'hola' that runs on Amiga 500!
